# Band Name Generator

## Overview

The **Band Name Generator** is a beginner-friendly Python project that creates a unique band name by combining the user's hometown and pet's name. This project demonstrates the fundamentals of Python, including user input, string manipulation, variables, functions, and formatted output.

---

## Concepts Used

* Functions
* Variables
* User Input
* String Manipulation
* String Formatting
* Formatted Output (f-strings)

---

## String Functions Used

### `strip()`

Removes leading and trailing spaces from a string.

**Example**

```python
name = "   john   "
print(name.strip())
```

**Output**

```text
john
```

---

### `title()`

Converts the first letter of every word into uppercase.

**Example**

```python
city = "new york"
print(city.title())
```

**Output**

```text
New York
```

---

## String Formatting (f-Strings)

The project uses Python's **f-string** feature to combine multiple values into one string.

**Example**

```python
name = "Alex"
print(f"Hello {name}")
```

**Output**

```text
Hello Alex
```
