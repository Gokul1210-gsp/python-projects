# Tip Calculator

## Overview

The **Tip Calculator** is a beginner-friendly Python project that calculates how much each person should pay after splitting a restaurant bill, including a selected tip percentage. This project demonstrates the fundamentals of Python, including user input, arithmetic operations, conditional statements, loops, exception handling, and formatted output.

---

## Learning Outcomes

After completing this project, you will understand how to:

* Create and call functions.
* Accept user input.
* Convert data using `int()` and `float()`.
* Perform arithmetic calculations.
* Use conditional statements and loops.
* Handle invalid input using exception handling.
* Apply Python string methods.
* Format output using f-strings.
* Build an interactive command-line application.

---

## Technologies Used

* Python 3
* Standard Python Libraries (No external packages required)
